# bevy_websocket_sync

Experimental real-time syncing code snippet for the Bevy Engine using WebSocket.

## Usage

Add `.env` as:

```
url=ws://localhost:8080
```

Then, start WebSocket server:

```
$ cd serve
$ node server
```

Finally, start client:

```
$ trunk serve
```
